# FlappyBird

使用pixi.js + gsap 实现经典游戏 FlappyBird

- pixi.js@v6.5.9

- gsap@3.7.0

## 启动

`http-server` 启用一个本地服务

```powershell
http-server --cors -c-1
```

## 功能

重新开始

历史高分

设备适配

资源加载
